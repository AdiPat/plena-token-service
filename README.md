# Setup

1. [Install Redis.](https://redis.io/docs/latest/operate/oss_and_stack/install/install-redis/)

2. Install dependencies.

```bash
npm install
```

# Start Server

1. Development server

```bash
npm run start:dev
```

2. Production build

```bash
npm run start
```

# Run Tests

1. Unit Tests

```bash
npm run test
```

1. Integration Tests

```bash
npm run test:e2e
```
